5. Groups of seletors


A comman-separated list of selector represents the union of all elements selected by each of the individual seletor in the list . (A comman is U+002C .).For example , in CSS when several seletor share the same declarations they may be grouped into a comma-separated list . White space may appear before and/or after the comma.


CSS example :


In the example , we condense three rules with identical declarations into one . Thus ,


h1 { font-family: sans-serif }

h2 { font-family: sans-serif }

h3 { font-family: sans-serif }


is equivalent to:


h1 , h2 , h3 { font-family: sans-serif }
